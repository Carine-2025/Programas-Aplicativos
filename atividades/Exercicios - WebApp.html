01
HTML — Estrutura
Montar o esqueleto de páginas e apps
EX 01 ⭐ Fácil
💡
Lógica: Toda página precisa da estrutura base: <html>, <head> (com <title>) e <body>. Dentro do body vão os elementos visíveis. Use <h1> para título e <p> para texto.
Cartão de visita simples
Crie uma página HTML com a estrutura completa contendo:
  • Título da página (aba do navegador): "Meu Cartão"
  • Seu nome completo em um <h1>
  • Sua profissão/curso em um <h2>
  • Seu e-mail em um <p>
  • Seu telefone em outro <p>
EX 02 ⭐ Fácil
💡
Lógica: Listas usam <ul> (não ordenada) ou <ol> (ordenada/numerada). Cada item vai dentro de um <li>. Para imagem use <img src="URL" width="200"> — a tag img não precisa fechar.
Página de receita
Crie uma página com uma receita culinária contendo:
  • Nome da receita em <h1>
  • Uma imagem do prato (pode usar qualquer URL de imagem da internet)
  • Subtítulo "Ingredientes" em <h2> seguido de uma lista não ordenada (<ul>)
  • Subtítulo "Modo de Preparo" em <h2> seguido de uma lista ordenada (<ol>)
EX 03 ⭐ Fácil
💡
Lógica: Links usam <a href="URL">texto</a>. Use <div> como caixa para agrupar elementos relacionados. Campos de formulário usam <input> com o atributo type (text, email, password) e placeholder para texto de dica.
Página de login
Crie uma tela de login com:
  • Um título <h1> escrito "Entrar"
  • Um <input type="email" placeholder="Seu e-mail">
  • Um <input type="password" placeholder="Sua senha">
  • Um <button> escrito "Entrar"
  • Um link <a> escrito "Esqueci minha senha"
EX 04 ⭐⭐ Médio
💡
Lógica: Tabelas usam <table>, com <tr> para cada linha, <th> para cabeçalho e <td> para dados. Pense na tabela como uma grade: cada <tr> é uma linha e cada <td> dentro dela é uma coluna.
Tabela de horários
Crie uma tabela com sua grade de horários da semana contendo:
  • Primeira linha com cabeçalhos: Horário, Segunda, Terça, Quarta, Quinta, Sexta
  • Pelo menos 3 linhas com matérias preenchidas
  • Coloque um <h1> acima da tabela escrito "Minha Grade"
EX 05 ⭐⭐ Médio
💡
Lógica: Use <div> para criar "caixas" que separam seções da página. <header> é a barra do topo, <main> é o conteúdo principal e <footer> é o rodapé. Essas tags semânticas funcionam como divs mas dizem o que cada parte é.
Página de produto
Crie uma página de loja virtual com:
  • Um <header> com o nome da loja em <h1>
  • Um <main> com: imagem do produto, nome em <h2>, descrição em <p>, preço em <p> e botão "Comprar"
  • Um <footer> com texto "© 2026 Minha Loja"

02
CSS — Visual
Estilizar, posicionar e deixar bonito
EX 06 ⭐ Fácil
💡
Lógica: O CSS fica dentro de <style> no <head>. Para mudar a aparência de todos os <h1>, escreva h1 { propriedade: valor; }. Use color para cor do texto, background para fundo, font-family para tipo de fonte e text-align: center para centralizar.
Estilizando o cartão de visita
Pegue o HTML do Exercício 01 e adicione CSS para:
  • Fundo da página em #1A1A2E (azul escuro)
  • Textos todos em white
  • Nome (h1) com tamanho 36px
  • Fonte da página toda: Arial
  • Tudo centralizado com text-align: center
  • Espaçamento interno da página: padding: 40px
EX 07 ⭐ Fácil
💡
Lógica: Para estilizar um elemento específico, dê a ele um id no HTML e use #id no CSS. Para estilizar vários elementos iguais, use class no HTML e .classe no CSS. Exemplo: <p class="destaque"> → .destaque { color: red; }
Cards coloridos
Crie 3 divs, cada uma representando um card, com:
  • Cada div com a classe "card"
  • CSS do .card: fundo branco, padding: 20px, margin: 10px, border-radius: 12px e sombra com box-shadow: 0 2px 8px rgba(0,0,0,0.1)
  • Cada card com um título (h3) e uma frase (p)
  • Dê um id diferente a cada card e mude a cor da borda esquerda com border-left: 4px solid COR
EX 08 ⭐⭐ Médio
💡
Lógica: Para estilizar botões, combine background, color, padding, border: none e border-radius. Para efeito ao passar o dedo/mouse, use button:hover { } e mude a cor de fundo. Para transição suave, adicione transition: 0.3s no botão normal.
Galeria de botões
Crie uma página com 4 botões diferentes, cada um com estilo próprio:
  • Botão 1: fundo azul, texto branco, cantos arredondados
  • Botão 2: sem fundo (transparente), borda verde de 2px, texto verde
  • Botão 3: fundo vermelho, texto branco, totalmente redondo (border-radius: 50px)
  • Botão 4: fundo preto, texto amarelo, todo em maiúsculas (text-transform: uppercase)
  • Todos devem mudar levemente de cor no :hover
EX 09 ⭐⭐ Médio
💡
Lógica: Para centralizar um bloco na tela, use max-width: 400px e margin: 0 auto. Para estilizar inputs, use width: 100%, padding, border, border-radius e font-size. Espaçamento entre elementos: margin-bottom.
Formulário bonito
Pegue o HTML do Exercício 03 (login) e estilize:
  • Centralize tudo com uma div de max-width: 360px e margin: 60px auto
  • Inputs com largura total, padding: 14px, borda cinza clara e cantos arredondados
  • Espaço entre cada campo: margin-bottom: 12px
  • Botão ocupando largura total com width: 100%, fundo azul e texto branco
  • Link "Esqueci minha senha" centralizado, cor cinza, sem sublinhado (text-decoration: none)
EX 10 ⭐⭐ Médio
💡
Lógica: Para colocar elementos lado a lado, use display: flex no elemento pai. Use gap para espaçamento entre filhos, flex-wrap: wrap para quebrar linha quando não couber, e justify-content: center para centralizar. Cada filho pode ter width: 200px fixo.
Galeria de fotos com Flexbox
Crie uma galeria de imagens usando Flexbox:
  • Um <h1> escrito "Minha Galeria"
  • Uma div com display: flex, flex-wrap: wrap e gap: 12px
  • Dentro, pelo menos 6 imagens (pode usar URLs de imagens da internet)
  • Cada imagem com width: 180px, height: 180px, object-fit: cover e border-radius: 10px

03
JavaScript — Comportamento
Interatividade, lógica e manipulação da página
EX 11 ⭐ Fácil
💡
Lógica: Use document.getElementById("id").innerText para mudar o texto de um elemento. Crie uma função com function nome() { } e conecte ela ao botão com onclick="nome()". Para pegar o valor digitado num input, use .value.
Saudação personalizada
Crie uma página com:
  • Um <input> com placeholder "Digite seu nome"
  • Um botão "Saudar"
  • Um <p id="saudacao"> vazio
  • Ao clicar no botão, o parágrafo deve mostrar: "Olá, [nome]! Bem-vindo(a)!"
EX 12 ⭐ Fácil
💡
Lógica: Para mudar cores com JS, use elemento.style.background = "cor". Você pode mudar a cor do document.body.style.background diretamente. Crie uma função para cada cor ou uma função que receba a cor como parâmetro: function trocar(cor) { }.
Troca-cor
Crie uma página com:
  • Um título "Escolha uma cor"
  • 3 botões: "Vermelho", "Azul", "Verde"
  • Ao clicar em cada botão, o fundo da página muda para a cor correspondente
  • Um 4° botão "Resetar" que volta o fundo para branco
EX 13 ⭐⭐ Médio
💡
Lógica: Para mostrar/esconder um elemento, use elemento.style.display = "none" (esconder) e elemento.style.display = "block" (mostrar). Você pode usar uma variável let visivel = true e alternar com if/else a cada clique.
Acordeão de perguntas (FAQ)
Crie um mini FAQ com:
  • 3 perguntas, cada uma em um <h3> com onclick
  • Cada resposta em um <p> que começa escondido (style="display: none")
  • Ao clicar na pergunta, a resposta aparece; clicar de novo, esconde
  • Estilize com CSS: fundo cinza claro em cada bloco, cantos arredondados
EX 14 ⭐⭐ Médio
💡
Lógica: Use um array (lista) para guardar itens: let lista = []. Adicione com lista.push(item). Para mostrar a lista na tela, use innerHTML e monte o HTML com um loop for ou lista.map(). Cada item vira um <li> dentro de um <ul>.
Lista de tarefas (To-Do)
Crie um app de tarefas com:
  • Um <input> para digitar a tarefa
  • Um botão "Adicionar"
  • Uma <ul id="tarefas"> onde as tarefas aparecem
  • Cada tarefa adicionada aparece como um novo <li> na lista
  • Se o input estiver vazio, mostre um alert("Digite uma tarefa!")
EX 15 ⭐⭐ Médio
💡
Lógica: Use Math.random() para gerar um número aleatório entre 0 e 1. Para um número de 1 a 10: Math.floor(Math.random() * 10) + 1. Use if/else para comparar o palpite do usuário com o número sorteado e dê feedback diferente para acertou, errou pra cima ou errou pra baixo.
Jogo de adivinhação
Crie um jogo onde o usuário tenta adivinhar um número:
  • Ao carregar a página, sorteie um número de 1 a 10
  • Um <input type="number"> para o palpite
  • Um botão "Tentar"
  • Um <p> que mostra: "Muito alto!", "Muito baixo!" ou "Acertou! 🎉"
  • Um contador de tentativas que mostra quantas vezes o jogador tentou
EX 16 ⭐⭐ Médio
💡
Lógica: Use setInterval(funcao, 1000) para executar algo a cada 1 segundo (1000 ms). Guarde os segundos numa variável e incremente. Para formatar em mm:ss, use Math.floor(seg / 60) para minutos e seg % 60 para segundos. Use clearInterval(id) para parar.
Cronômetro
Crie um cronômetro com:
  • Um <h1> mostrando "00:00"
  • Botão "Iniciar" — começa a contar os segundos
  • Botão "Parar" — pausa o cronômetro
  • Botão "Zerar" — volta para 00:00
  • Estilize com CSS: números grandes, botões coloridos
EX 17 ⭐⭐ Médio
💡
Lógica: Para a calculadora de IMC: IMC = peso / (altura * altura). Use .toFixed(1) para mostrar só 1 casa decimal. Use if/else if para classificar: abaixo de 18.5, entre 18.5-24.9, entre 25-29.9, acima de 30. Mude a cor do resultado conforme a faixa.
Calculadora de IMC
Crie uma calculadora de Índice de Massa Corporal:
  • Input para peso (kg) e input para altura (m)
  • Botão "Calcular"
  • Exiba o resultado com a classificação: Abaixo do peso, Normal, Sobrepeso ou Obesidade
  • Mude a cor do texto do resultado: verde para Normal, amarelo para Sobrepeso, vermelho para Obesidade
  • Estilize todo o formulário com CSS bonito
EX 18 ⭐⭐⭐ Desafio
💡
Lógica: Combine tudo que aprendeu. Use um array para guardar as tarefas, localStorage.setItem("chave", JSON.stringify(array)) para salvar a lista inteira e JSON.parse(localStorage.getItem("chave")) para recuperar. Para remover, use array.splice(indice, 1) e re-renderize a lista.
App de tarefas completo (com localStorage)
Evolua o Exercício 14 para um app completo:
  • Adicionar tarefas com input + botão
  • Cada tarefa tem um botão "✕" que a remove
  • As tarefas são salvas no localStorage — ao fechar e abrir o navegador, elas continuam lá
  • Estilize como um app real: fundo claro, cards com sombra, botões bonitos
  • Mostre um contador: "3 tarefas pendentes"
EX 19 ⭐⭐⭐ Desafio
💡
Lógica: Use async function e await fetch(url) para buscar dados. A API ViaCEP é gratuita: https://viacep.com.br/ws/CEP/json/. Substitua CEP pelo valor digitado. A resposta JSON terá campos como logradouro, bairro, localidade, uf. Trate erros com if (!resposta.ok).
Buscador de CEP
Crie um app que consulta endereços:
  • Um input para digitar o CEP (só números)
  • Um botão "Buscar"
  • Ao clicar, busque o endereço na API ViaCEP
  • Exiba: rua, bairro, cidade e estado em parágrafos separados
  • Se o CEP não existir, mostre "CEP não encontrado" em vermelho
  • Estilize como um app com CSS
EX 20 ⭐⭐⭐ Desafio
💡
Lógica: Este exercício junta HTML + CSS + JS + localStorage + API. Monte a estrutura HTML, estilize com CSS como se fosse um app real (dark mode ou light mode, sua escolha), use JS para a lógica e localStorage para persistir os dados. Pense em cada parte separadamente e depois conecte tudo.
🏆 Projeto Final — Mini App de clima
Crie um app completo de previsão do tempo:
  • Input para digitar o nome de uma cidade
  • Botão "Buscar clima"
  • Use a API gratuita: https://wttr.in/CIDADE?format=j1
  • Exiba: temperatura atual, sensação térmica e descrição do tempo
  • Salve a última cidade pesquisada no localStorage e carregue automaticamente ao abrir
  • Estilize como um app real com ícones (use emojis: ☀️🌧️⛅), cores e cantos arredondados

🚀 Dica final: Comece sempre pelo HTML (estrutura), depois CSS (visual) e por último JS (comportamento).
